# Puzzle Challenge
 
Run in Android:
  1) yarn install
  2) yarn run android
Run in Ios:
  1) yarn install
  2) cd ios
  3) pod install
  4) open in xcode the challenge.xcworkspace file located in ios folder
  5) in xcode press build and run.
